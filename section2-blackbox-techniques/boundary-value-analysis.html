Boundary Value Analysis

Boundary Value Analysis (BVA) is a black-box testing technique that focuses on testing the boundaries of equivalence partitions. Boundary values are more likely to expose defects compared to values within the partitions.

Real-life Example: Age Validation

In the form below, you can enter an age. The valid range for age is between 18 and 65. Use Boundary Value Analysis to test values at the boundaries (e.g., 17, 18, 65, 66) to see if the system accepts or rejects them appropriately.

Explanation of Boundary Values:

  • Valid Boundaries: Ages 18 and 65 are valid boundary values.
  • Invalid Boundaries: Ages below 18 and above 65 (e.g., 17 and 66) are invalid boundary values.
