Decision Table Testing

Decision Table Testing is a black-box test technique that helps you test different combinations of inputs and their corresponding outputs. It is especially useful when the system has complex logic with multiple conditions leading to different outcomes.

Real-life Example: Online Discount System

Consider an online store that provides discounts based on the user's membership and the total cart value. Use the form below to select a membership type and cart value, then see how the system applies discounts based on the decision table logic.
