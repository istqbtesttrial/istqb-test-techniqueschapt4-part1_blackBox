Welcome to ISTQB Foundation: Test Techniques

This course will teach you to understand and apply a variety of software test techniques.

Test techniques help the tester in test analysis (what to test) and in test design (how to test). These techniques are divided into three categories:

  • Black-box testing: Focuses on the behavior of the system without knowledge of internal structures.
  • White-box testing: Relies on knowledge of the internal structure to design tests.
  • Experience-based testing: Leverages the tester’s experience to detect defects.

White-box Testing Overview

White-box testing analyzes the internal structure and processing of the system. You will often test all possible paths and logic in the code.

Code Snippet: Path Coverage

Look at this function and identify test cases to ensure 100% path coverage:

          function sum(arr) {
            let total = 0;
            for (let i = 0; i < arr.length; i++) {
              total += arr[i];
            }
            return total;
          }
